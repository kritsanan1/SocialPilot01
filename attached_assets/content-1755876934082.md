Available on Business, Enterprise plans.

Specify a User Profile in an API call by [adding the Profile Key in the header](https://www.ayrshare.com/docs/apis/overview#profile-key-format).

User Profiles are one of the core concepts of Ayrshare that allows you to create and manage multiple users. Each one of the users of your platform will have one or more Ayrshare User Profiles. Each User Profile can have one connection to every supported social network.

## [​](https://www.ayrshare.com/docs/apis/profiles/overview\#profile-key)  Profile Key

Many endpoints, such as /user, /analytics, or /delete, can be called on behalf of a Profile account by [adding the “Profile-Key” parameter](https://www.ayrshare.com/docs/apis/overview#profile-key-format) in the Header.For example, the /delete endpoint can be called to delete a Profile account’s post for the given _post_ _id_ for the provided _Profile Key_.

```
const API_KEY = "API_KEY";
const PROFILE_KEY = "PROFILE_KEY";
const id = "Post ID";

fetch("https://api.ayrshare.com/api/delete", {
      method: "DELETE",
      headers: {
        "Content-Type": "application/json",
        "Authorization": `Bearer ${API_KEY}`
        "Profile-Key": PROFILE_KEY
      },
      body: JSON.stringify({
        id: id
      }),
    })
      .then((res) => res.json())
      .then((json) => console.log(json))
      .catch(console.error);

```

## [​](https://www.ayrshare.com/docs/apis/profiles/overview\#enable-or-disable-social-networks)  Enable or Disable Social Networks

You can disable and enable social networks at either the global level or at the User Profile level.

- [Globally manage social account access](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles#set-social-networks-access) in the dashboard.
- Manage social account access for a User Profile when [creating](https://www.ayrshare.com/docs/apis/profiles/create-profile) or [updating](https://www.ayrshare.com/docs/apis/profiles/update-profile).

It is important to note that disabled social networks in the global setting, set via the dashboard, override User Profile settings.For example, if you disable TikTok access in the dashboard, then no User Profile can access TikTok. On the other hand, if you turn on TikTok access in the dashboard but disable it for a specific User Profile, all User Profiles will have access to TikTok except that particular one.

## [​](https://www.ayrshare.com/docs/apis/profiles/overview\#get-the-linked-social-accounts)  Get the Linked Social Accounts

Retrieve the social accounts a User Profile has linked with either the [/profiles endpoint](https://www.ayrshare.com/docs/apis/profiles/get-profiles) or the [/user endpoint](https://www.ayrshare.com/docs/apis/user/profile-details).

## [​](https://www.ayrshare.com/docs/apis/profiles/overview\#post-to-a-user-profile)  Post to a User Profile

With the [/post](https://www.ayrshare.com/docs/apis/post/post) endpoint, you can post to a User Profile by adding the Profile Key in the header. The return will be an array of posts or error results, with an overall status. If all posts successful, “success”, or if any post failed, “error”.You may obtain a user’s Profile Key when the profile is created with the [/create-profile](https://www.ayrshare.com/docs/apis/profiles/create-profile) endpoint or in Ayrshare’s Web Dashboard GUI by switching to the profile and going to **Profile Key** page.
